# Blue_Archive_VideoTextBlur
烤肉机part1，去除文字

当前进度：

- [x] openCV-inpaint 实现单张图片去除文字
- [x] pysubs2 处理字幕标记时间轴
- [x] 视频去除文字并输出
- [x] 字幕样式
- [x] 多线程
- [x] 声音
- [x] 字幕视频压制
- [x] 字幕文本换行
- [x] 进度条
